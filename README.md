# Nexus ⚡

Nexus is a scalable, real-time communication platform built to handle instant messaging efficiently. Developed using the MERN stack and Socket.io, it focuses on backend architecture, secure authentication, and seamless event-driven state management.

### Core Features
* **Real-Time Messaging:** Instant 1-on-1 and group chats powered by Socket.io.
* **Secure Authentication:** JWT-based login and registration using modern `HttpOnly` cookie strategies for session management.
* **Optimized Architecture:** Relational NoSQL database design for fast room creation and message retrieval.
* **Tech Stack:** MongoDB, Express.js, React.js, Node.js.
